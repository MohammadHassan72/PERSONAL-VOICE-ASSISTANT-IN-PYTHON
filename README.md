# Personal Voice Assistant in Python

This is a simple Python project implementing a personal voice assistant that recognizes voice commands and performs various tasks accordingly.

## Features

- **Voice Recognition**: Utilizes the `speech_recognition` library to recognize voice commands.
- **Text-to-Speech Conversion**: Employs the `pyttsx3` library to convert text into speech for interactive responses.
- **Task Execution**: Executes tasks based on recognized voice commands such as opening specific websites, playing songs on YouTube, and more.

## Requirements

- Python 3.x
- Required Python libraries: `speech_recognition`, `pyttsx3`, `pywhatkit`
- Internet connection (for tasks like opening websites)

## Installation

1. Clone or download the project repository to your local machine.
2. Install the required Python libraries using pip:
   ```
   pip install speech_recognition pyttsx3 pywhatkit
   ```

## Usage

1. Run the `main.py` file.
2. Ensure your microphone is connected and properly set up.
3. Upon running, the assistant will listen for voice commands.
4. Speak clearly and wait for the assistant to process your command.
5. The assistant will respond accordingly, executing tasks like opening websites, playing songs, etc.

## Supported Commands

- **Greeting**: Say "Hi" to trigger a greeting message from the assistant.
- **Open YouTube**: Command the assistant to open YouTube.
- **Play Song**: Ask the assistant to play a song (currently defaults to a random Arijit Singh song).
- **Open WhatsApp**: Instruct the assistant to open WhatsApp Web.
- **Search Google**: Command the assistant to perform a Google search.
- **Open Wikipedia**: Instruct the assistant to open a specific Wikipedia page.
- **Open Facebook**: Command the assistant to open Facebook.
- **Open Chat-GPT**: Instruct the assistant to open Chat-GPT from OpenAI.
- **Open Instagram**: Command the assistant to open Instagram.
- **Open Zomato**: Instruct the assistant to open Zomato.
- **Open Amazon**: Command the assistant to open Amazon.
